## Summary

Cloud Architect and CTO with 25+ years of experience designing, building, and scaling payment platforms, serverless systems, and B2B SaaS products. Proven track record as a technical cofounder, solo architect, and engineering leader. Deep expertise in AWS, PHP, TypeScript, Terraform, and REST API design.

## Core Competencies

- **Cloud Architecture**: AWS (Lambda, CloudFormation, CDK, Cognito, S3, CloudFront, Aurora, VPC), Terraform, Infrastructure as Code
- **Backend Engineering**: PHP (Laminas/Mezzio), TypeScript, Node.js, Python, REST API design, OAuth2
- **Payment Systems**: PCI-compliant payment processing, recurring billing, merchant self-service platforms
- **DevOps & CI/CD**: Docker, GitHub Actions, Bitbucket Pipelines, Blue/Green deployments, LocalStack
- **Data & Storage**: PostgreSQL (PostGIS), MySQL (Aurora), Redis, OpenSearch, Prisma ORM
- **Frontend**: Next.js, React, React Native, Turborepo monorepo architecture
- **Leadership**: Technical cofounder (2x), CTO, team lead, architectural decision-making, mentorship

Architecting and building a verification-first B2B SaaS review platform for the wellness industry from the ground up.

- Designed full-stack TypeScript monorepo architecture (Next.js 16, React 19, Turborepo, Prisma ORM, PostgreSQL)
- Provisioned all cloud infrastructure via Terraform: WorkOs, AWS S3, CloudFront, Cloudflare DNS/WAF, Neon PostgreSQL, Vercel
- Led research across 6 SaaS infrastructure domains; authored 14 architectural decision records

### Lead Architect & DevOps Engineer — Globalized Mobile Platform (2021–present)

Led DevOps and infrastructure architecture for a mature mobile platform built as a microservices monorepo. 43,964+ commits over 4 years.

- Architected Blue/Green deployment system with 7-phase pipeline, automated rollback, and zero-downtime orchestration across CloudFormation, Serverless Framework, and SAM
- Achieved >70% reduction in AWS costs through infrastructure optimization
- Led complete PostgreSQL migration from MySQL Aurora + OpenSearch Serverless, tuned for 2,000+ concurrent connections

Cofounded and built GetCheddar, the first true SaaS recurring billing platform with fully automated merchant self-signup and self-service.

- Wrote 99% of the codebase from PHP5/Zend Framework v1 through PHP7/Mezzio rewrite as pure REST API with OAuth2
- Migrated from Rackspace Cloud to proprietary PCI-compliant hosting environment with zero downtime
- Specialized in usage tracking and billing automation for web application subscriptions

### CTO & Cofounder — Resite Information Technology (2000–2007)

Cofounded and built the first automated payment acceptance platform for the US multi-family housing industry.
